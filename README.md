# UIDAI Aadhaar Data Analysis – Maharashtra

This project was developed for the UIDAI Data Hackathon 2026.
It analyzes Aadhaar enrolment, demographic updates, and biometric updates
across Maharashtra using Python.

## Objectives
- Identify enrolment patterns
- Analyze district-wise performance
- Compare urban vs rural activity
- Provide policy recommendations

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Maharashtra has 94%+ Aadhaar saturation
- Shift from enrolment to maintenance phase
- Urban centers handle majority updates
- Rural access gap exists

## Outputs
- Analytical graphs
- District & area-wise insights
- Actionable recommendations
